# feedme — 요구사항 정의서

## 핵심 아이디어

URL 하나만 넣으면 웹페이지나 YouTube 콘텐츠를 마크다운으로 추출해서 클립보드에 복사하여, 어떤 AI 챗봇에든 바로 붙여넣을 수 있는 미니멀 웹 도구.

## 사용자 흐름

1. 웹 앱에 접속하면 URL 입력창이 보인다
2. 웹페이지 또는 YouTube URL을 붙여넣고 추출 버튼을 클릭한다
3. 콘텐츠가 마크다운으로 변환되어 미리보기로 표시된다
4. 클립보드 복사 버튼을 클릭한다
5. 원하는 AI 챗봇에 직접 붙여넣는다

## 설계 원칙

- 편집 없이 바로 복사 — 불필요한 중간 단계를 없앤다
- API 키 불필요 — 클립보드 복사만으로 동작하므로 설정이 필요 없다
- 챗봇 비종속 — 공통 마크다운 형식 하나로 어떤 AI 챗봇에든 사용 가능하다
