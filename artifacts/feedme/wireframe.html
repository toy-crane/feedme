웹 접근성 가이드라인 소개

웹 접근성은 장애 여부와 관계없이 모든 사람이 웹 콘텐츠를 이용할 수 있도록 보장하는 것을 의미합니다.

주요 원칙

  • 인식의 용이성
  • 운용의 용이성
  • 이해의 용이성
  • 견고성
복사됨
Notes

웹페이지 콘텐츠가 마크다운으로 렌더링된 결과 화면.

  • 트리거: 웹페이지 URL 추출 완료
  • 인터랙션: '복사' 버튼 클릭 → 마크다운 원문 클립보드 복사
  • 전환: 복사 클릭 → '복사됨' 토스트 표시 후 유지, 새 URL 입력 → 다시 추출 가능

Notes

추출 실패 시 에러 메시지를 표시하는 화면.

  • 트리거: 빈 입력/잘못된 URL/접근 불가/자막 없음
  • 에러 메시지 종류:
    • 빈 입력, 잘못된 형식 → "올바른 URL을 입력해주세요"
    • 접근 불가 → "페이지에 접근할 수 없습니다"
    • 자막 없음 → "자막을 찾을 수 없습니다"
  • 인터랙션: URL 수정 후 다시 '추출' 클릭
  • 전환: 유효 URL 재입력 + 추출 → 로딩 화면
