## Overview
현재 단순한 "복사" 버튼을 shadcn의 DocsCopyPage 패턴처럼 split button + dropdown으로 업그레이드하여, 클립보드 복사 외에 ChatGPT/Claude에서 바로 열기 기능을 제공한다.

## 시나리오

### 1. 복사 버튼 기본 동작 (기존 유지)
[상황] 콘텐츠가 추출되어 미리보기가 표시된 상태
[동작] 메인 "복사" 버튼을 클릭하면 마크다운이 클립보드에 복사되고 "복사됨" 토스트가 표시된다

성공 기준:
- [ ] 복사 아이콘 + "복사" 텍스트가 메인 버튼에 표시된다
- [ ] 클릭 시 마크다운이 클립보드에 복사된다
- [ ] 복사 후 아이콘이 체크 아이콘으로 바뀌고 "복사됨" 토스트가 표시된다

### 2. 드롭다운 트리거
[상황] 콘텐츠가 추출되어 복사 버튼이 표시된 상태
[동작] 메인 버튼 옆 chevron 버튼을 클릭하면 드롭다운 메뉴가 열린다

성공 기준:
- [ ] 메인 버튼 오른쪽에 chevron(▼) 아이콘 버튼이 표시된다
- [ ] chevron 클릭 시 드롭다운 메뉴가 열린다
- [ ] 드롭다운에 "ChatGPT에서 열기", "Claude에서 열기" 2개 항목이 표시된다
- [ ] 각 항목에 서비스 아이콘이 함께 표시된다

### 3. ChatGPT에서 열기
[상황] 드롭다운 메뉴가 열린 상태
[동작] "ChatGPT에서 열기"를 클릭하면 마크다운 내용을 query로 포함한 ChatGPT URL이 새 탭에서 열린다

성공 기준:
- [ ] 클릭 시 `https://chatgpt.com/?q={인코딩된 마크다운}` URL이 새 탭(_blank)에서 열린다

### 4. Claude에서 열기
[상황] 드롭다운 메뉴가 열린 상태
[동작] "Claude에서 열기"를 클릭하면 마크다운 내용을 query로 포함한 Claude URL이 새 탭에서 열린다

성공 기준:
- [ ] 클릭 시 `https://claude.ai/new?q={인코딩된 마크다운}` URL이 새 탭(_blank)에서 열린다

### 5. 콘텐츠 없을 때 버튼 미표시 (기존 유지)
[상황] 콘텐츠가 추출되지 않은 초기 화면
[동작] 화면을 확인한다

성공 기준:
- [ ] 복사 split button이 표시되지 않는다

## 범위

### 포함
- 기존 "복사" 버튼을 split button + dropdown 패턴으로 교체
- 드롭다운에 ChatGPT, Claude 열기 항목 추가
- 각 서비스별 SVG 아이콘 표시

### 제외
- View as Markdown 옵션 — feedme는 이미 마크다운을 보여주므로 불필요
- v0, Scira 등 다른 서비스 — 사용자 요청에 따라 ChatGPT, Claude만 지원
- 모바일 전용 Popover 분기 — 변경 비용이 작으므로 기본 DropdownMenu로 통일

## 전제 조건
- shadcn DropdownMenu 컴포넌트 설치
- shadcn Separator 컴포넌트 (이미 설치됨)
