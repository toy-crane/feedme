## Overview
추출된 콘텐츠를 AI에 보내거나 복사할 때, 사용자가 원하는 지시문(프롬프트)을 앞에 붙일 수 있게 한다. 접이식 UI로 기본은 숨겨져 있고, 필요할 때만 펼쳐서 커스텀 프롬프트를 입력하거나 프리셋을 선택한다.

## 시나리오

### 1. 프롬프트 영역 표시
[상황] 콘텐츠가 추출되어 미리보기가 표시된 상태에서
[동작] 제목과 복사 버튼 사이에 접힌 상태의 "프롬프트 추가" 토글이 표시된다

성공 기준:
- [ ] 콘텐츠 추출 완료 -> Collapsible 트리거("프롬프트 추가")가 표시됨
- [ ] 콘텐츠 미추출 초기 화면 -> Collapsible 트리거가 표시되지 않음
- [ ] Collapsible 기본 상태 -> 접힘(닫힘)

### 2. 프롬프트 영역 펼치기/접기
[상황] "프롬프트 추가" 토글을 클릭하면
[동작] Textarea 입력창과 프리셋 ToggleGroup이 펼쳐진다

성공 기준:
- [ ] 접힌 상태에서 트리거 클릭 -> Textarea와 ToggleGroup 3개 칩이 표시됨
- [ ] 펼쳐진 상태에서 트리거 클릭 -> Textarea와 ToggleGroup이 숨겨짐

### 3. 프리셋 선택
[상황] 프롬프트 영역이 펼쳐진 상태에서 프리셋 칩을 클릭하면
[동작] 해당 프리셋 텍스트가 Textarea에 채워진다

성공 기준:
- [ ] "요약해줘" 칩 클릭 -> Textarea에 "요약해줘" 텍스트가 채워짐
- [ ] "한국어로 번역해줘" 칩 클릭 -> Textarea에 "한국어로 번역해줘" 텍스트가 채워짐
- [ ] "쉽게 설명해줘" 칩 클릭 -> Textarea에 "쉽게 설명해줘" 텍스트가 채워짐
- [ ] 이미 선택된 칩을 다시 클릭 -> Textarea가 비워지고 칩 선택 해제

### 4. 커스텀 프롬프트 입력 시 칩 선택 해제
[상황] 프리셋 칩으로 Textarea가 채워진 상태에서 사용자가 텍스트를 수정하면
[동작] ToggleGroup의 선택 상태가 자동으로 해제된다

성공 기준:
- [ ] "요약해줘" 칩 선택 후 텍스트를 "요약해줘 한국어로" 로 수정 -> 칩 선택 해제됨
- [ ] 사용자가 직접 Textarea에 입력 (칩 선택 없이) -> 칩은 선택되지 않은 상태 유지

### 5. 프롬프트와 함께 복사
[상황] 프롬프트가 입력된 상태에서 "복사하기" 버튼을 클릭하면
[동작] "프롬프트\n\n마크다운" 형식으로 클립보드에 복사된다

성공 기준:
- [ ] 프롬프트 "요약해줘" + 마크다운 "# Hello" -> 클립보드에 "요약해줘\n\n# Hello" 복사됨
- [ ] 프롬프트 비어 있음 + 마크다운 "# Hello" -> 클립보드에 "# Hello"만 복사됨 (기존 동작)

### 6. 프롬프트와 함께 AI에서 열기
[상황] 프롬프트가 입력된 상태에서 "ChatGPT에서 열기" 또는 "Claude에서 열기"를 클릭하면
[동작] "프롬프트\n\n마크다운"이 query에 포함된 URL이 새 탭에서 열린다

성공 기준:
- [ ] 프롬프트 "요약해줘" + "ChatGPT에서 열기" -> URL query에 "요약해줘\n\n마크다운" 인코딩됨
- [ ] 프롬프트 "요약해줘" + "Claude에서 열기" -> URL query에 "요약해줘\n\n마크다운" 인코딩됨
- [ ] 프롬프트 비어 있음 + "ChatGPT에서 열기" -> 마크다운만 query에 포함 (기존 동작)

### 7. 마크다운 다운로드 시 프롬프트 미포함
[상황] 프롬프트가 입력된 상태에서 "마크다운 다운로드"를 클릭하면
[동작] 마크다운 원문만 다운로드된다 (프롬프트 미포함)

성공 기준:
- [ ] 프롬프트 "요약해줘" + "마크다운 다운로드" -> 다운로드 파일에 마크다운만 포함, 프롬프트 없음

### 8. 로고 클릭 시 프롬프트 초기화
[상황] 프롬프트가 입력된 상태에서 로고를 클릭하면
[동작] URL, 결과, 에러, 프롬프트가 모두 초기화된다

성공 기준:
- [ ] 프롬프트 "요약해줘" 입력 상태에서 로고 클릭 -> 프롬프트 비워짐, Collapsible 접힘, ToggleGroup 선택 해제

## 범위

### 포함
- Collapsible 접이식 프롬프트 입력 영역
- Textarea를 통한 커스텀 프롬프트 입력
- ToggleGroup 프리셋 3개: 요약해줘, 한국어로 번역해줘, 쉽게 설명해줘
- 복사/ChatGPT/Claude 전송 시 프롬프트 합침
- 로고 클릭 시 프롬프트 초기화

### 제외
- 프롬프트 히스토리 저장 — MVP 범위 외
- 사용자 정의 프리셋 추가/삭제 — MVP 범위 외
- 마크다운 다운로드 시 프롬프트 포함 — 다운로드는 원본 보존 목적
