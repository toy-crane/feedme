## Overview

웹페이지나 YouTube URL을 입력하면 콘텐츠를 마크다운으로 추출하여 클립보드에 복사할 수 있는 미니멀 웹 도구. 추출한 마크다운을 어떤 AI 챗봇에든 바로 붙여넣을 수 있다.

## 시나리오

### 1. 웹페이지 콘텐츠 추출

[상황] URL 입력창이 비어 있는 초기 화면에서
[동작] 웹페이지 URL을 입력하고 '추출' 버튼을 클릭하면, 콘텐츠가 마크다운으로 변환되어 렌더링된 미리보기로 표시된다

성공 기준:
- [ ] 웹페이지 URL 입력 후 '추출' 클릭 → 제목과 본문이 렌더링된 마크다운 미리보기로 표시
- [ ] 추출 진행 중 로딩 인디케이터가 표시된다

### 2. YouTube 콘텐츠 추출

[상황] URL 입력창이 비어 있는 초기 화면에서
[동작] YouTube URL을 입력하고 '추출' 버튼을 클릭하면, 영상 정보(제목, 채널명, 설명)와 자막이 마크다운으로 추출되어 렌더링된 미리보기로 표시된다

성공 기준:
- [ ] YouTube URL 입력 후 '추출' 클릭 → 제목, 채널명, 설명, 자막이 렌더링된 마크다운으로 표시
- [ ] 추출 진행 중 로딩 인디케이터가 표시된다

### 3. 초기 상태에서 복사 버튼 비노출

[상황] 콘텐츠가 추출되지 않은 초기 화면에서
[동작] 화면을 확인하면 '복사' 버튼이 표시되지 않는다

성공 기준:
- [ ] 초기 화면에서 '복사' 버튼이 보이지 않는다

### 4. 클립보드 복사

[상황] 콘텐츠가 추출되어 미리보기가 표시된 상태에서
[동작] '복사' 버튼을 클릭하면 마크다운 원문이 클립보드에 복사되고 '복사됨' 피드백이 표시된다

성공 기준:
- [ ] '복사' 버튼 클릭 → 클립보드에 마크다운 텍스트가 복사된다
- [ ] 복사 완료 후 '복사됨' 토스트 메시지가 표시된다

### 5. 빈 URL로 추출 시도

[상황] URL 입력란이 비어 있는 상태에서
[동작] '추출' 버튼을 클릭하면 에러 메시지가 표시된다

성공 기준:
- [ ] 빈 입력 상태에서 '추출' 클릭 → '올바른 URL을 입력해주세요' 에러 메시지 표시

### 6. 잘못된 URL 처리

[상황] URL 입력창에 잘못된 형식의 텍스트가 입력된 상태에서
[동작] '추출' 버튼을 클릭하면 에러 메시지가 표시된다

성공 기준:
- [ ] "not-a-url" 입력 후 '추출' 클릭 → '올바른 URL을 입력해주세요' 에러 메시지 표시

### 7. 접근 불가 URL 처리

[상황] URL 입력란에 접근 불가능한 URL이 입력된 상태에서
[동작] '추출' 버튼을 클릭하면 에러 메시지가 표시된다

성공 기준:
- [ ] 접근 불가 URL 입력 후 '추출' 클릭 → '페이지에 접근할 수 없습니다' 에러 메시지 표시

### 8. 자막 없는 YouTube 영상

[상황] 자막이 없는 YouTube 영상의 URL이 입력된 상태에서
[동작] '추출' 버튼을 클릭하면 자막 없음 에러 메시지가 표시된다

성공 기준:
- [ ] 자막 없는 YouTube URL 입력 후 '추출' 클릭 → '자막을 찾을 수 없습니다' 에러 메시지 표시

## 범위

### 포함

- URL 입력 및 추출 버튼
- 웹페이지 본문 마크다운 변환 (defuddle)
- YouTube 영상 정보 + 자막 추출 (defuddle)
- 렌더링된 마크다운 미리보기
- 클립보드 복사 및 피드백
- 에러 처리 (잘못된 URL, 접근 불가, 자막 없음)

### 제외

- 콘텐츠 편집 기능 — 설계 원칙 '편집 없이 바로 복사'
- 추출 히스토리 저장 — MVP 범위 외
- API 키 입력/관리 — 설계 원칙 'API 키 불필요'
- 챗봇별 맞춤 형식 — 설계 원칙 '챗봇 비종속'
